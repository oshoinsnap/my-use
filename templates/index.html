FAQ
Which column is used for de-duplication?

By default use the column containing emails (case-insensitive, trims whitespace). If multiple email columns exist, pick the first non-empty.

What file types are supported?

Currently .xlsx. Convert CSV/Google Sheets to .xlsx before uploading.

Is the upload secure?

Implement HTTPS on your server and delete processed files after generating the refined download.
